О проекте

Это современный шаблон для разработки фронтенд-приложений, сочетающий статические HTML-страницы с интерактивными Vue компонентами.

Технологии:

  • Vite 7 - быстрая сборка
  • Vue 3 - реактивные компоненты
  • TypeScript - типизация
  • SCSS - мощные стили
  • Handlebars - шаблоны HTML
